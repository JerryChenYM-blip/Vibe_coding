## 🚀 如何使用

### 1. 啟動應用程式
```bash
cd /Users/jerrychen/project/Claude_code
venv/bin/python3 main.py
```

應用程式視窗將出現，包含：
- **頂部欄位** — 模型選擇器 (目前為：large-v3-turbo)，語言選擇器 (目前為：自動偵測)
- **錄音區域** — 波形視覺化 + 圓形錄音按鈕
- **結果區域** — 轉錄文字顯示
- **操作按鈕** — 複製、儲存、Ollama、設定

### 2. 測試錄音 (3 種方法)

#### 方法 A：使用 GUI 按鈕
1. 點擊中心處的綠色圓形按鈕
2. 說出您的訊息 (可以包含英文和中文)
3. 完成後點擊按鈕或按鈕將返回綠色
4. 結果將顯示在按鈕下方

#### 方法 B：使用全局熱鍵 ⌘⇧Space
1. 在 Mac 的任何地方**按住** ⌘⇧Space
2. 說出您的訊息
3. **放開**按鍵以停止錄音
4. 轉錄將自動開始

#### 方法 C：使用系統設定測試
如果您跳過了輔助功能權限：
1. 點擊 ⚙ (設定) → 尋找輔助功能連結
2. 或開啟系統設定 → 隱私權與安全性 → 輔助功能
3. 將終端機 (或 Python) 加入清單
4. 重新啟動應用程式

### 3. 要測試的功能

- **中英混合** — 嘗試："你好，我在學習 Python 和 JavaScript"
- **自動語言偵測** — 應用程式會自動偵測語言
- **複製按鈕** — 將轉錄內容複製到剪貼簿
- **儲存按鈕** — 儲存為 .txt 檔案
- **設定面板** — 更改模型、語言、熱鍵

## 📝 疑難排解

### 問題：應用程式視窗未出現
**解決方案：**
- 確保您使用的是虛擬環境的 Python：`venv/bin/python3`
- 檢查日誌：`tail -50 app.log`
- 嘗試從終端機啟動：`cd /Users/jerrychen/project/Claude_code && venv/bin/python3 main.py`

### 問題：全局熱鍵無效
**解決方案：**
1. 檢查輔助功能權限：
   - 系統設定 → 隱私權與安全性 → 輔助功能
   - 確保終端機或 Python 在清單中
2. 加入權限後重新啟動應用程式
3. 驗證：啟動時應用程式應顯示綠色的確認資訊

### 問題：轉錄速度慢或不準確
**解決方案：**
1. 確保您使用的是 `large-v3-turbo` 模型 (更適合中英混合)
2. 確保麥克風音質良好
3. 說話清晰，句子之間有自然的停頓
4. 如果效能有問題，請先嘗試 `small` 模型

### 問題：模型未下載
**解決方案：**
1. 刪除快取：`rm -rf ~/.cache/huggingface/hub/`
2. 重新啟動應用程式 — 它將重新下載模型
3. 首次下載約需 5-10 分鐘，視連線狀況而定
